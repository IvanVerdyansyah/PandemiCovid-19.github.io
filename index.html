Pendidikan Ring Wabah Virus Corona

dibuat oleh ivanverdyansyah. pada tanggal 2 november 2021

Pandemi COVID-19 ngawenang dampak ring makesami kauripan manusa, rumasuk ring pendidikan. Kawentenan wabah virus corona puniki ngawenang pialang ring palaksanaan paplajahan sane biasane tatap muka mangkin dados sistem daring (dalam jaringan). Anggen ngelidin pandemi COVID-19 puniki, pemerintah sampun ngawentenang awig-awig mangda sekolah-sekolah nunas sisianyane malajah ring jero. Ngawit pinanggal 16 Maret 2020 sekolah nincapang metode malajah daring. Punika manut ring awig-awig Menteri Pendidikan lan Kebudayaan RI ngeninin indik sewalapatra nomor 4 warsa 2020 indik awig-awig pendidikan ring parindikan corona virus desease (COVID-19) puniki.

Ngawit sewalapatra kajalanin punika SMAN 1 Singaraja sampun nincapang sistem paplajahan online utawi daring. Ri kala sistem punika kakawitin sekolah nenten biasa nganggen paplajahan daring ngawinang wenten pialang ring guru lan sisia. Ring guru wenten makudang pialang sane karasayang meweh rikala nyiapang media paplajahan sane becik mangda sisia nenten med malajah ring jero, guru taler nenten uning indik tata cara makarya media paplajahan online. Punika mawinan wenten pikobet ring paplajahan sisia sane ngaranayang sisia lengit ngamiletin paplajahan krana nenten waged ring materi sane kaicen olih guru. Ngawitin paplajahan daring ring SMAN 1 Singaraja puniki guru kari nganggen akeh aplikasi rikala ngicenin media paplajahan ngawinang para sisia akeh sane madue pikobet rikala makarya tugas nganggen makudang aplikasi media paplajahan online sakadi Edmodo, Google Classroom, Schoology lan aplikasi sane lianan. Makudang sisia pedih krana ponsel sane kagelahang naenin pikobet rikala kaanggen ngirim tugas krana akehnyane aplikasi sane kaanggen olih soang-soang guru. Lianan punika, sisia taler ngarasa repot olih tugas sekolah sane sukeh lan galah kapupulan tugas punika kawatesin, yadiastuan ipun durung waged indik materi sane kaicenin olih guru.

Lantas ring warsa paplajahan anyar 2020/2021 sistem daring SMA N 1 Singaraja sampun kauwah nganggen pamekas paum para guru indik pikobet-pikobet sane karasayang, mangkin paplajahan daring wantah nganggen asiki aplikasi kemanten inggih punika Google Classroom. Punika mawinan sisia nenten malih mengunduh akeh aplikasi sane malianan ring paplajahan. Akeh guru taler sampun ngembangang media paplajahan sane pacang kaicenin ring para sisia dados becik marupa video wiadin power point. Lianan punika guru lan sisia taler polih paket internet gratis saking pemerintah sabilang sasih mangda paplajahan lancar. Punika mawinan guru lan sisia aluh ngirim materi majeng ring sisia utawi mupulang tugas sane kaicen ring guru. Krana rikala nenten polih paket internet gratis saking pemerintah makudang sisia wiadin guru sukeh krana paplajahan online nganggen akeh paket internet. Keberhasilan guru rikala ngalaksanayang paplajahan daring ring unduk pandemi COVID-19 puniki inggih punika kawagedan guru berinovasi ngarencanayang, lan ngaryanin materi, metode paplajahan, lan aplikasi napi sane manut sareng materi lan metode. Kreatifitas punika kunci sukses saking guru mangda prasida memotivasi sisianyane teleb malajah daring (online) lan nenten dados beban psikis.

Punika mawinan, paplajahan daring dados solusi sane becik anggen iraga para sisia malajah, mangda prasida megatin penyebaran COVID-19 ring wewidangan sekolah, physical distancing utawi menjaga jarak aman taler dados pertimbangan paplajahan jarak jauh utawi daring kapilih. Kerjasama sane becik ring guru, sisia, rerama sisia lan pihak sekolah dados faktor penentu mangda paplajahan daring punika prasida luwih. Dumadak pandemi covid-19 puniki enggal ical rikala new normal sampun kalaksanayang pemerintah. Ngawinang paplajahan prasida mamargi sakadi ipidan rikala guru lan sisia saling matepuk.
